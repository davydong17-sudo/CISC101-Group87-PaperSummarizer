Each module defines a specific part of the summarization pipeline.

## How the System Works
1. User provides paper text + settings  
2. Module 01 validates inputs  
3. Module 02 loops through each section and generates summaries  
4. Module 03 applies guardrails (strict evidence mode + missing/short section warnings)  
5. Module 05 extracts direct citations  
6. Module 06 improves readability  
7. Module 04 assembles the final output
